
# Lotto Number Generator

## Overview

A simple, stylish web application to generate random lottery numbers.

## Project Outline

*   **HTML (`index.html`):** The main structure of the application, including the title, a container for the numbers, and a button to generate them.
*   **CSS (`style.css`):** Modern and bold styling for a visually appealing and responsive user interface. This will include a gradient background, custom fonts, and interactive button styles.
*   **JavaScript (`main.js`):** The core logic for generating unique random lottery numbers and updating the display. It will use a Web Component to display the generated numbers.
